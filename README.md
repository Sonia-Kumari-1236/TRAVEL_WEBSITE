#  Travel Website – Frontend Project

A modern, responsive **Travel Website** built using **HTML, CSS, and JavaScript**.  
This project showcases destinations, travel blogs, testimonials, and interactive UI elements, inspired by real-world travel platforms.

The main focus of this project is **clean UI design**, **smooth animations**, and a **user-friendly experience**.

---

##  Features

-  **Hero & Destination Sections** with high-quality visuals  
-  **Travel Blog Slider** using Swiper.js  
-  **Testimonials Section** with user reviews  
-  **Animated Static Counter** for bookings & tours  
-  **Video Background Section** for immersive experience  
-  Clean layout with a modern color palette 


---

##  Technologies Used

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox, Grid, transitions & overlays  
- **JavaScript (ES6)** – Interactions & animations  
- **Swiper.js** – Sliders & carousels  
- **Font Awesome** – Icons
